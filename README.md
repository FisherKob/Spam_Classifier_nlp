# Spam_Classifier_nlp

In this module, I implemented NLP techniques such as stemming, CountVectorizer to preprocess the dataset and build a Naive Bayes Classifier to predict whether a given email is a spam or not.

# Info about dataset

The SMS Spam Collection is a set of SMS tagged messages that have been collected for SMS Spam research. It contains one set of SMS messages in English of 5,574 messages, tagged acording being ham (legitimate) or spam. More information can be found in [Kaggle] (https://www.kaggle.com/uciml/sms-spam-collection-dataset)
